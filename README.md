自動破解數獨的演算法，分成「fast mode」與「pro mode」。只要將此演算法套用至任一份數獨題目，就會自動生成答案。
fast mode：可利用同區域消去（同行、同列、同方格檢查重複數字）
pro mode：可利用stack資料結構實作猜數字系統，將答案猜出來。

輸入格式：
第一行：2或3，表示數獨一行共有幾個大方格/n
第二行：fast 或 pro
fast 為沒有猜數字系統的基礎模式
pro 為擁有猜數字系統的進階模式
第三行：輸入整篇數獨
數字間沒有空格，若原本該格沒有數字，填入0

輸出格式：
多次檢查用除錯訊息
整篇填滿的數獨，數字間以空格隔開
